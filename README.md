# 🚗 Old Car Price Prediction App

A machine learning-based **Streamlit web application** that predicts the price of used/old cars based on their features like brand, model, year, mileage, fuel type, and more.

---

##  Project Objective

The goal of this project is to help users estimate the fair market price of a used car using a trained machine learning model.

---

##  Features

-  Predict used car price instantly
-  Input car details (brand, year, mileage, fuel type, etc.)
-  Machine Learning model-based prediction
-  Clean and modern Streamlit UI
-  Responsive sidebar navigation
-  Fast and interactive results

---

##  Machine Learning

This project uses a trained regression model to predict car prices based on historical data.

Possible algorithms used:
- Linear Regression / Ridge / Lasso
- Random Forest Regressor (if applicable)

- Python 🐍
